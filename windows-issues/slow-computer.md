# Slow Windows Computer Troubleshooting Guide

## 📌 Issue
User reports that the computer is extremely slow during startup and when opening applications.

## 👤 Environment
- OS: Windows 11
- Device: Corporate laptop
- User Type: Standard user

## 🚨 Possible Causes
- Too many startup programs
- Low RAM
- Malware infection
- High disk usage
- Pending Windows updates

## 🔍 Diagnostic Steps

1. Open Task Manager (Ctrl + Shift + Esc)
2. Check CPU and Memory usage.
3. Navigate to the Startup tab.
4. Identify high-impact startup apps.
5. Check disk usage.
6. Run Windows Defender scan.

## 🛠 Resolution

✅ Disabled unnecessary startup applications  
✅ Cleared temporary files using Disk Cleanup  
✅ Ran malware scan  
✅ Installed pending updates  
✅ Restarted device  

## ✅ Outcome
System performance improved significantly. Boot time reduced from 5 minutes to under 1 minute.

## 📚 Prevention
- Limit startup apps
- Schedule regular updates
- Run weekly malware scans
